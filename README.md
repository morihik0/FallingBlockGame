# FallingBlockGame

・ゲーム制作コミュニティで連休中にゲームを制作するという企画で制作した物   
・Unity5.3.3で制作（制作期間1日）   
・Audio/font関連のファイルは抜いてあります。   
・ゲームの実行ファイルは/Buildにあります（win7/osx10.5.5で動作確認）   

・ゲーム操作   
下部の黒いバーを操作   
←→で移動してスペースキーで上にある受け皿を入れ替える   

3種類の具材が繋がれば鍋になって+1点　5点毎にレベルアップして速度アップ   
★は触れた列の具材を消去する   

ゲームスタート、リトライはｚキー   

音/絵素材はフリー素材です。
